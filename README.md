# To-Do-List-Web-App
<h1>Objective</h1>
A simple and interactive To-Do List application built with HTML, CSS, and Vanilla JavaScript.This task demonstrates the fundamentals of DOM manipulation, event handling, and dynamic UI updates.

# Workflow
1. **Set up the basic HTML structure**

   * Created an input field for entering tasks with Id "taskInput".
   * Added an **Add** button with Id "addBtn".
   * Included a list container to display tasks with Id "taskList".

2. **Style the layout with CSS**

   * Applied CSS to create a clean and organized user interface.
   * Improved spacing, alignment, and overall readability.

3. **Add event listeners using JavaScript**

   * Attached an event listener to the **Add** button.
   * Triggered task creation when the button is clicked.

4. **Append new tasks dynamically**

   * Created new list items (`<li>`) using JavaScript.
   * Added each new task to the task list without reloading the page.

5. **Implement task completion**


   * Enabled users to mark tasks as complete.
   * Enabled users to mark tasks as complete by Underline the task it showa that the task is completed.
   * Used the `classList.toggle()` method to toggle a CSS class for completed tasks.

6. **Add task removal functionality**

   * Added a **Remove/Delete** button for each task.
   * Allowed users to delete tasks instantly from the list.

7. **Test application functionality**

   * Verified that tasks can be added successfully.
   * Confirmed that completed tasks toggle correctly.
   * Ensured tasks are removed as expected.

8. **Provide instant UI updates**

   * Ensured all changes are reflected immediately.
   * Performed all updates dynamically using DOM manipulation without requiring a page reload.

**Technologies Used**
* HTML5
* CSS3
* Vanilla JavaScript 

**What I Learned**
* DOM manipulation
* Event handling using addEventListener()
* Creating and appending HTML elements dynamically
* Toggling CSS classes
* Removing DOM elements
* Building an interactive web application using Vanilla JavaScript
